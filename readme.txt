Add professional typographic endings to your content with magazine-style symbols.

== Description ==

Endmark adds a trailing symbol or image to the end of your posts and pages, giving your content a polished, editorial appearance.

= Professional Features =
* Choose between custom symbols or images
* Magazine-style article endings
* Works on posts, pages, or both
* Responsive design friendly
* French translation included
* Easy-to-use settings panel

= Typographic Excellence =
Add that final flourish to your articles with:
- Classic typographic symbols (#, †, §, etc.)
- Custom images/logos
- Perfect for long-form content
- Mobile-responsive display

== Installation ==

1. Upload the `endmark` folder to `/wp-content/plugins/`
2. Activate through *Plugins → Installed Plugins*
3. Configure under *Settings → Endmark*
4. Choose between symbol/image display
5. Select display locations (posts/pages/both)

== Frequently Asked Questions ==

= Can I use custom fonts for symbols? =
Yes! Use any character supported by your theme's font.

= How do I add my company logo? =
Use the image uploader in settings to select any PNG/JPG/SVG file.

= Is this compatible with page builders? =
Yes, works with Gutenberg, Elementor, and other editors.
